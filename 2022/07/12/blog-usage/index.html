    博客使用及维护的说明

    暂未分类
    发布时间 : 2022-07-12 23:49
    阅读 :

      1. 新建文章
      2. 本地运行服务
      3. 生成静态文件
      4. 发布博客

    博客使用 Hexo! 构建。更多细节可查阅 Hexo 的详细 文档 .

    使用的主题为 3-hex。 更多使用方法查阅作者个人 blog 的相关文章。

    本文主要摘录上面两者中常用的使用方法，用以备忘。

    Hexo 命令

    新建文章

    $ hexo new "My New Post"

    More info: Writing

    本地运行服务

    $ hexo server

    More info: Server

    生成静态文件

    $ hexo generate

    More info: Generating

    发布博客

    $ hexo deploy

    More info: Deployment
