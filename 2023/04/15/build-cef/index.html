    编译 cef

    开源库 编译 构建 cef
    发布时间 : 2023-04-15 17:02
    阅读 :
    评论:

    编译环境：Debian10
    cef版本：5060

    1. 设置 cef 构建根目录, 如 cef_build, 并将该目录设置为环境变量 CEF_BUILD_ROOT_DIR
      我这里使用的是 ~/Workspace/cef_build，该目录需要使用绝对路径。
      如果使用下面命令，则相对路径和绝对路径均可。（按实际替换命令中的 ~/Workspace/cef_build）
    export CEF_BUILD_ROOT_DIR=$(realpath -e ~/Workspace/cef_build)
    1. 在根目录下创建两个目录 automate 和 chromium_git
      命令如下
    mkdir $CEF_BUILD_ROOT_DIR/automate && mkdir $CEF_BUILD_ROOT_DIR/chromium_git
    1. 下载并运行脚本
      这里使用 curl 下载，如果没有该命令则需先自行安装。

    下载运行脚本执行下面的命令即可

    cd $CEF_BUILD_ROOT_DIR
curl 'https://chromium.googLesource.com/chromium/src/+/refs/heads/main/build/install-build-deps.sh?format=TEXT' | base64 -d > install-build-deps.sh
chmod 755 install-build-deps.sh
sudo ./install-build-deps.sh --no-arm --no-chromeos-fonts --no-nacl

    这个链接下载不下来：https://chromium.googlesource.com/chromium/src/+/master/build/install-build-deps.sh?format=TEXT

    1. 下载工具包
      执行如下命令开始下载
    cd $CEF_BUILD_ROOT_DIR
git clone https://chromium.googlesource.com/chromium/tools/depot_tools.git
    1. 添加工具包到环境变量 PATH
    export PATH=$CEF_BUILD_ROOT_DIR/depot_tools:$PATH
    1. 下载自动化脚本 automate-git.py
    cd $CEF_BUILD_ROOT_DIR/automate
wget https://bitbucket.org/chromiumembedded/cef/raw/master/tools/automate/automate-git.py
    1. 创建拉取（更新）代码的脚本
    echo '#!/bin/bash' > $CEF_BUILD_ROOT_DIR/chromium_git/update.sh
echo 'python ../automate/automate-git.py --download-dir=$CEF_BUILD_ROOT_DIR/chromium_git --depot-tools-dir=$CEF_BUILD_ROOT_DIR/depot_tools --branch=5060 --no-distrib --no-build' >> $CEF_BUILD_ROOT_DIR/chromium_git/update.sh
cd $CEF_BUILD_ROOT_DIR/chromium_git
chmod 755 update.sh
./update.sh
