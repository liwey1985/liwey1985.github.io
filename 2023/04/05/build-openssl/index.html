    开源库 openssl 的构建

    开源库 openssl 编译 构建
    发布时间 : 2023-04-05 15:15
    阅读 :
    评论:

    官网：https://www.openssl.org
    源码：https://github.com/openssl/openssl
    版本：1.1.1t

    假定源码目录为 OPENSSL_SRC_DIR

    Linux 下构建：

    cd $OPENSSL_SRC_DIR
mkdir build && cd build
mkdir installed
../Configure linux-x86_64 --prefix=$OPENSSL_SRC_DIR/build/installed --openssldir=$OPENSSL_SRC_DIR/build/installed --api=1.0.0
make && make install

    说明， 这里构建默认的动态库，如果要构建静态库库，则 Configure 后面的选项需要加上 no-shared。如果该静态库要被其它动态库链接则还要加上 -fPIC，已避免链接时报错。
    --api=1.0.0 用于兼容 1.0.0版本的api。 由于之前有别的代码用的openssl时1.0版本的，故加上该选项。
