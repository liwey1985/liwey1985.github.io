    开源库 zlib 的构建

    开源库 编译 构建 zlib
    发布时间 : 2023-04-05 12:03
    阅读 :

    官网：http://www.zlib.net/
    源码：https://github.com/madler/zlib
    版本：1.2.13

    假定源码目录为 ZLIB_SRC_DIR

    Linux 下构建：

    cd $ZLIB_SRC_DIR
mkdir build && cd build
cmake .. -DCMAKE_INSTALL_PREFIX=$ZLIB_SRC_DIR/build/installed
make && make test
make install

    构建完成后，可以发现 installed/lib 下面同时包含有静态和动态库
